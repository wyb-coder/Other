## 🚀 运行步骤

### 1. 进入项目目录

```bash
cd /path/to/Experiment1/code/1.2
```

### 2. 编译

```bash
make
```

### 3. 运行测试

**终端 1 - 启动服务器：**

```bash
./tcp_server
```

**终端 2 - 启动客户端：**

```bash
./tcp_client
```

客户端输入消息，服务器接收后回显。输入 `quit` 或 `exit` 退出。

## 🔧 常见问题

| 问题       | 解决方案                          |
| ---------- | --------------------------------- |
| 端口被占用 | 修改 `SERVER_PORT` 或等待一段时间 |
| 连接被拒绝 | 先启动服务器，再启动客户端        |

---

## 📝 TCP vs UDP 区别

| 特性       | TCP                     | UDP                       |
| ---------- | ----------------------- | ------------------------- |
| 连接方式   | 面向连接（需三次握手）  | 无连接                    |
| 可靠性     | 可靠传输                | 不保证                    |
| 服务器函数 | `listen()` + `accept()` | 直接 `recvfrom()`         |
| 数据传输   | `send()` / `recv()`     | `sendto()` / `recvfrom()` |
